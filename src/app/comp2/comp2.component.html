Angular Components

Components are the main building block for Angular applications.

Components define views, which are sets of screen elements that Angular can choose among and modify according to your program logic and data.

Generating a component

Generate a new component by the command: ng generate component my_comp_name

By default, this command will generate the following files:

  • A folder named after the component
  • A component file, my_comp_name.component.ts
  • A template file, my_comp_name.component.html
  • A CSS file, my_comp_name.component.css
  • A testing specification file, my_comp_name.component.spec.ts
