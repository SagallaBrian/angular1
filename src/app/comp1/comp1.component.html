What is angular ?

Angular is an application design framework and development platform for creating efficient and sophisticated single-page apps.

Angular is a development platform, built on TypeScript

The official website for Angular: Angular

Setting up Angular

  1. To install Angular for your computer run the command: npm install -g @angular/cli.
  2. Note in linux and mac Os due to some root permission you need to add sudo hence sudo npm install -g @angular/cli
  3. Generating a new workspace: ng new my_project_name
  4. Start the application ng serve or run the command ng serve --open
